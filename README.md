# Simple Kanban Board

A basic, client-side Kanban board application built with HTML, CSS, and vanilla JavaScript.

## Features

*   **Columns:** "To Do", "In Progress", and "Done" columns to organize tasks.
*   **Add Tasks:** Easily add new tasks to the "To Do" column using a button and a prompt.
*   **Drag & Drop:** Move tasks between columns using drag-and-drop functionality.
*   **Basic Styling:** Clean and simple interface.
*   **Unit Tests:** Includes basic QUnit tests for core functionality.

## How to Use

1.  Clone or download the repository.
2.  Open the `index.html` file in your web browser.
3.  To run the tests, open the `tests.html` file in your web browser.

## Project Structure

*   `index.html`: The main HTML file for the Kanban board.
*   `style.css`: Contains all the styles for the application.
*   `script.js`: Handles the JavaScript logic for task creation and drag-and-drop.
*   `tests.html`: The HTML runner for QUnit tests.
*   `tests.js`: Contains the QUnit tests.
*   `README.md`: This file.
